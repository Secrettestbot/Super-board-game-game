# Super Board Game Game

A comprehensive collection of classic board games playable in the terminal.

## Features
- 20+ classic board games with multiple variations
- Integrated tutorials for every game
- Suspend and resume games at any time
- Game variations for added replayability
- Two-player local play

## Games Included
- Chess (Standard, Chess960, King of the Hill, Three-Check)
- Checkers/Draughts (American, International, Brazilian, Turkish)
- Go (9x9, 13x13, 19x19)
- Reversi/Othello
- Backgammon
- Mancala (Kalah, Oware, Congkak)
- Nine Men's Morris (+ Six Men's, Three Men's, Twelve Men's)
- Connect Four (Standard, Five-in-a-Row/Connect Five, Pop Out)
- Tic-Tac-Toe (3x3, 4x4, 5x5, Ultimate)
- Gomoku (Standard, Renju)
- Battleship
- Snakes and Ladders
- Ludo
- Nim (Standard, Misère)
- Dots and Boxes
- Hex
- Pong Hau K'i
- Alquerque
- Fox and Hounds
- Tablut (Viking Chess)

## Running
```bash
python3 main.py
```

## Requirements
- Python 3.7+
- No external dependencies required
